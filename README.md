# QuantumCore
Computador FM foi um método usado nos anos 1980 para transmitir programas de computador pelo rádio FM.  A ideia era simples:  a rádio enviava dados digitais convertidos em áudio (FSK) pelo canal estéreo;  o ouvinte gravava o sinal em fita cassete usando um rádio;  depois carregava o programa no computador (MSX, Spectrum etc.) como se fosse uma fita
